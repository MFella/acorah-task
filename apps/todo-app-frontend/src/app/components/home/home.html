TodoApp is a modern Task Manager application built as an advanced recruitment project (SaaS Blueprint).

💡 System Architecture:

  • Frontend: Angular (Signals, Standalone Components, Tailwind CSS)
  • Backend: NestJS (Monorepo, REST API, Class-Validator)
  • Database: SQLite managed by TypeORM

The application fully supports relational one-to-many relationships (User ➔ Todo Lists ➔ Tasks) and ensures complete timezone independence by storing dates in UTC format (ISO 8601).
